# 旅行工具 (TravelTool)

一个基于纯 HTML + CSS 构建的旅游平台网站，提供目的地浏览、酒店查询、旅游攻略等功能。

## 项目简介

旅游平台致力于为旅行者提供优质的旅游信息服务，包括热门目的地推荐、精选酒店展示、旅游攻略分享等功能模块。网站采用多页面静态架构，无需构建工具，可直接在浏览器中运行。

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **原生 JavaScript** - 交互逻辑

## 快速开始

### 方式一：直接打开
直接在浏览器中打开 `src/index.html` 文件即可预览网站。

### 方式二：使用本地服务器（推荐）

```bash
# 进入项目目录
cd TravelTool

# 使用 Python 启动本地服务器
python -m http.server 8080

# 或使用 Node.js 的 http-server
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080/src/index.html`

## 开发说明

- 所有页面共享统一的头部导航和页脚设计
- 样式文件按页面独立拆分，便于维护
- 使用 CSS 变量统一管理主题色彩（建议后续添加）
- 图片资源使用外部占位图服务
